# Drivers Hub: Backend

This is the backend for Drivers Hub made by Charles.  
External plugins supported, add files to [external_plugins](/src/external_plugins/) and edit config, then they'll be loaded without the need to compile again!

**Main functions:**  
1.Discord OAuth2 Login + Steam & TruckersMP Account Bind  
2.Delivery log and tracker powered by Truckspace/Navio  
3.Role-based permission management  

**Main plugins:**  
1.Announcement  
2.Application  
3.Division  
4.Downloads  
5.Event
